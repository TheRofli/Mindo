# Mindo Roadmap

Mindo's near-term roadmap is focused on making the plugin reliable, reviewable, and useful in ordinary vault work.

## Now

- Keep the Community Plugin install small and self-contained.
- Maintain clean release checks for public GitHub and Obsidian review.
- Improve vault-aware chat, source references, and safe note actions.
- Keep generated sidecars and local runtime caches out of the public source tree.

## Next

- Improve Mindo Wiki memory quality, repair flows, and maintenance prompts.
- Strengthen local voice setup for STT/TTS without making it required for basic chat.
- Continue refining Mindo Code planning workflows inside Obsidian.
- Expand runtime diagnostics so users can understand which optional local features are installed.

## Later

- Add more platform-specific runtime packaging paths.
- Make local acceleration easier to install and verify.
- Broaden workflow automation while preserving explicit user control over vault edits.
