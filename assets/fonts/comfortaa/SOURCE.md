# Comfortaa font source

Comfortaa Regular is bundled from the official Google Fonts repository.
Google Fonts currently distributes Comfortaa as a variable TTF; the bundled
file is used at regular weight 400 in the plugin UI.

- Font file: https://github.com/google/fonts/blob/main/ofl/comfortaa/Comfortaa%5Bwght%5D.ttf
- License file: https://github.com/google/fonts/blob/main/ofl/comfortaa/OFL.txt
- Specimen page: https://fonts.google.com/specimen/Comfortaa

The font is distributed under the SIL Open Font License, Version 1.1.
